\section{\makebox[\widthof{\faUsers}][c]{\color{CVBlue}\faUsers}\ 项目与研究经历}
% ========== 新增：葡萄膜炎诊断项目 (当前研究) ==========
\textbf{基于多模态大模型的葡萄膜炎诊断研究} \hfill 2025.9 -- 至今
\titlebreak
项目描述：研究多模态大模型在眼科疾病（葡萄膜炎）诊断中的应用，旨在实现高精度、可解释的自动化诊断，并探索模型的数据效率、模态贡献与领域泛化能力。
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{模型与数据}：采用 \textbf{Qwen3-VL-8B-Instruct} 模型，在包含1834张眼底图像（894名患者）的私有数据集上进行 \textbf{LoRA微调}。采用患者级别的严格数据划分，避免数据泄露。
    \item \textbf{高性能基线}：通过系统的超参数调优，模型在独立测试集上达到了\textbf{95.74\%}的准确率与\textbf{96.52\%}的F1分数，为后续消融实验建立了强基线。
    \item \textbf{系统消融分析}：主导设计了数据量影响、模态贡献等多组消融实验。首次在该任务上量化了训练数据量（5\%-100%）与模型性能的关联，并评估了FFA、OCT、Optos三种成像模态的诊断价值。
    \item \textbf{前沿对比}：与多个零样本医学模型（\textbf{FLAIR}, \textbf{PubMedCLIP}）及眼科专用基础模型（\textbf{RETFound}）进行对比，系统分析了微调大模型相对于领域专用模型及零样本方法的优势。
\end{itemize}

\textbf{基于知识图谱与LLM的医疗问答系统} \hfill 2023.03 -- 2023.07
\titlebreak
项目描述：作为核心开发成员，参与构建一个结合医疗知识图谱与大语言模型（LLM）的智能问答系统，旨在提供精准、可解释的医疗信息查询服务。
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
\item 负责系统的\textbf{对话管理}与\textbf{查询转换模块}。将用户的自然语言问题解析并转换为面向知识图谱的结构化查询（如Cypher语句）。
\item 利用\textbf{Python}及相关NLP库（如spaCy）处理用户输入，并结合规则与关键词匹配初步识别医疗实体与意图。
\item 设计并实现了与\textbf{大语言模型（如ChatGPT API）}的协同流程：先由LLM对复杂查询进行意图澄清与信息补全，再将优化后的问题送入知识图谱引擎，最后整合两者的返回结果生成友好回复。
\item 该项目实践了\textbf{RAG (检索增强生成)} 的基本思想，并锻炼了将前沿LLM能力与传统知识库系统结合落地的工程能力。
\end{itemize}
